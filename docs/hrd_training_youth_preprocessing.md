# HRD 훈련과정 청년 적합 전처리 문서

## 1. 사용 컬럼

원본 HRD 데이터에서 다음 컬럼을 중심으로 전처리했습니다.

- `title`: 훈련과정명
- `subTitle`: 훈련기관명
- `trainTarget`: 훈련대상
- `address`: 지역
- `traStartDate`, `traEndDate`: 훈련기간
- `courseMan`, `realMan`: 훈련비 또는 지원 정보
- `certificate`: 관련 자격증
- `ncsCd`: NCS 코드
- `titleLink`: 상세 URL
- `titleIcon`, `contents`: 훈련 유형/설명 보조 텍스트

## 2. raw_text 생성

RAG 검색을 위해 훈련과정명, 기관, 대상, 지역, 기간, 비용, 자격증, NCS 코드, 상세 URL을 합쳐 `raw_text`를 생성했습니다.

## 3. youth_relevance 기준

### HIGH

청년에게 직접적으로 적합하거나 청년 취업에 유용한 과정입니다.

- 청년 직접 키워드 포함
- 디지털/IT/AI/데이터/개발 관련 과정
- 구직자/미취업자/취업준비/취업연계/양성과정 성격이 강한 과정

### MEDIUM

청년도 수강 가능하지만 청년 특화 근거가 약한 일반 훈련 과정입니다.

- 일반 국민내일배움카드 과정
- 일반 자격증 과정
- 회계/세무/조리/요양보호/기능사 등 일반 직업훈련

### LOW

청년 지원 서비스와 관련성이 낮거나 기업/사업주/중장년 중심 과정입니다.

- 사업주훈련
- 기업 단체훈련
- 법정의무교육
- 관리자/임원 교육
- 중장년/시니어/고령자 중심 교육

## 4. training_type 기준

아래 유형으로 분류했습니다.

- IT/디지털
- 데이터/AI
- 개발
- 디자인/UX
- 사무/회계
- 서비스
- 보건/복지
- 제조/기술
- 조리/식음료
- 기타

## 5. 통합 전 주의사항

이번 패키지는 정제까지만 수행합니다.
기존 `opportunities.json`과의 통합은 다음 단계에서 진행해야 합니다.

서비스에 우선 붙일 파일은 `training_youth_high_only.json`입니다.
RAG에 우선 사용할 파일은 `training_youth_high_chunks.jsonl`입니다.
